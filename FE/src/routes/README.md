# Routes Configuration

This folder contains the routing configuration for the application.

## Routes

The main routes are defined in `index.js` and include:
- `/` - Home page
- `/cars` - Cars listing and browsing
- `/my-bookings` - User bookings management

## Usage

Import routes in App.jsx:
```jsx
import { Routes, Route } from 'react-router-dom'
import routes from './routes'
```
